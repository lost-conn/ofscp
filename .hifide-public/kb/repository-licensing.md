## License Choice

**MIT License** selected for maximum adoption:
- Permissive: Allows commercial/proprietary use.
- Simple attribution.
- Standard for specs/protocols (e.g., similar to ActivityPub influences).

**Rationale**:
- Repo focus: OFSCP spec/docs → Encourage implementations.
- No code yet → Avoid copyleft (GPL).
- Alternatives: Apache 2.0 (patents), but MIT simpler.
