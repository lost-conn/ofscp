# OFSCP Specification Overview

The Open Federated Social Communications Protocol (OFSCP) is defined in `docs/ofscp_spec.md`.

## Key Features
- OAuth 2.0 for authentication.
- Federation for groups, channels, and DMs.
- Privacy tiers for channels.
- No End-to-End Encryption; security via auth tokens.

## Updates
- Removed E2EE and encryption guidelines.
- Uses Bearer tokens for API security.

Refer to the full spec for details.
